#Bottle Todo List 

A todo list made with Bottle for studies purposes.

#Getting started



###Create a virtualenv

```sh
$ virtualenv todo
$ source todo/bin/activate
```


### Install the dependencies
```sh
$ pip install -r requirements.txt
```

### Create the Database
```sh
$ python init-db.py
```

### Run the app
```sh
$ python todo.py
```
